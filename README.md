# Akmatori - AI-Powered Operations Assistant

> **Coming Soon**

Akmatori is an intelligent AIOps platform that leverages [OpenAI Codex CLI](https://github.com/openai/codex) to automate incident investigation, troubleshooting, and operational tasks.

## Overview

Akmatori connects your monitoring systems with AI-powered agents that can:

- Automatically investigate alerts and incidents
- Execute diagnostic commands and analyze results
- Provide intelligent recommendations for remediation
- Integrate with your existing tools and workflows

## Key Features

- **Slack Integration** - Interact with AI agents directly from Slack
- **Prometheus/PagerDuty/Grafana/Zabbix Integration** - Automatic alert ingestion and investigation
- **Custom Agents** - Define specialized agents for different operational domains
- **Web Dashboard** - Monitor investigations and manage configurations
- **Extensible Tools** - Add custom scripts and tools for your environment

## Architecture

- **Backend**: Go
- **Frontend**: React + TypeScript
- **Database**: PostgreSQL
- **AI Engine**: Powered by Codex CLI
